# Cookagent product plan

## Vision
Cookagent is a mobile-first AI-powered recipe management PWA for personal recipe storage, ingestion, reading, and later menu planning.

## MVP
- Store and read recipes.
- Local-first database.
- Offline viewing.
- Manual recipe editing.
- Paste-text ingestion.
- Screenshot/book-photo ingestion.
- LLM extraction into structured recipe draft.
- Review before save.
- Tags and filters.
- Printable recipe page.
- Human-readable export/import backup.

## Not MVP
- YouTube ingestion.
- Nutrition.
- Google Drive sync.
- Recipe chat.
- Semantic search.
- Duplicate detection.
- Version history.
- Voice input.
- Meal calendar.
- Shopping list.

## Preferred architecture
- Next.js App Router
- TypeScript
- Tailwind
- Dexie / IndexedDB
- Zod
- LLM provider abstraction for local model first, OpenAI later
